# White-Matter

This repository contains the archives created for a project of the subject called Medical Imaging of the degree in Telecommunications Engineering. 
This project aims to address the segmentation of white matter in magnetic resonance imaging of the brain. <br /> <br />

## Requirements
The code is written in MATLAB R2021b. You can execute the code in this MATALB version or later.

## Instructions

### 1. Read the pdfs about the State-of-Art in white matter segmentation and those about the deliverables of different stages.

### 2. Download the repository with the subject and the MATLAB code.

### 3. Execute the code in MATLAB. You can also use ITK-SNAP to segment the white matter.
